ASCII Art Library Project

Objective: Develop a reusable C++ library for generating ASCII art, implement advanced loop structures, and write the output to a file using command-line arguments.

Description

This project focused on creating a library of functions for ASCII art generation. The library supports patterns like rectangles and triangles. A demonstration program was built to showcase the library's capabilities.

Key Functions

// Draw a Rectangle
void drawRectangle(int width, int height, char fill, char border, std::ostream &out) {
    for (int i = 0; i < height; ++i) {
        for (int j = 0; j < width; ++j) {
            if (i == 0 || i == height - 1 || j == 0 || j == width - 1) {
                out << border;
            } else {
                out << fill;
            }
        }
        out << std::endl;
    }
}
            
            
// Write ASCII Art to File
bool writeToFile(const char *filename, const std::string &content) {
    std::ofstream file(filename);
    if (!file.is_open()) return false;
    file << content;
    file.close();
    return true;
}

Challenges and Solutions

  • Managing primitive arrays: Required careful memory handling and boundary checks.

  • File handling: Implemented robust parsing for command-line arguments.
